PHP 7.0.9 is Released
The PHP development team announces the immediate availability of PHP 7.0.9. This is a security release. Several security bugs were fixed in this release, including the HTTP_PROXY issue. All PHP 7.0 users are encouraged to upgrade to this version.

The PHP development team announces the immediate availability of PHP 7.1.0 Beta 1. This release is the first beta for 7.1.0. All users of PHP are encouraged to test this version carefully, and report any bugs and incompatibilities in the bug tracking system.

Interesting Projects, Tools and Libraries


promises
Promises/A+ implementation that handles promise chaining and resolution iteratively, allowing for "infinite" promise chaining while keeping the stack size constant.

scrapbook
PHP caching environment, with adapters for e.g. Memcached, Redis, Couchbase, APC, SQL and additional capabilities (e.g. transactions, stampede protection) built on top.

phulp
Task manager for PHP, the PHP port of Gulp.

retrofit-php
This library aims to ease creation of REST clients. It is blatantly stolen from square/retrofit and implemented in PHP.

keenclient-php
Official PHP client for the Keen IO API. Build analytics features directly into your PHP apps.

browniephp
BrowniePHP is a plugin for the CakePHP framework that saves you the work to create a back-end for your app.

color-thief-php
A PHP class for grabbing the color palette from an image. Uses PHP and GD or Imagick libraries to make it happen.

php-ansible
This library is an oop-wrapper for the ansible provisioning tool.

colorjizz-php
ColorJizz is a PHP library for manipulating and converting colors.

php-bignumbers
A robust library to handle immutable big numbers inside PHP applications.

chat-api
The PHP WhatsApp library.

sylius
eCommerce PHP framework built on top of Symfony with component-based architecture and format-agnostic rendering.
